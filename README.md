# Broadcast CMS

Hệ thống quản lý nội dung phát thanh truyền hình (Broadcast Content Management System).

## Tính năng

- Quản lý đề tài (Topics)
- Quản lý quy trình sản xuất (Workflow)
- Thống kê và báo cáo (Dashboard & Analytics)
- Phân quyền người dùng theo vai trò (Role-based Access Control)

## Công nghệ sử dụng

- **Frontend:** React, Vite, TailwindCSS
- **UI Library:** Shadcn UI
- **State Management:** Zustand
- **Routing:** TanStack Router
- **Charts:** Recharts

## Cài đặt và chạy

1. Clone dự án
```bash
git clone https://github.com/TruongDann/broadcast-cms.git
```

2. Cài đặt dependencies
```bash
npm install
```

3. Chạy server phát triển
```bash
npm run dev
```
